# readme-generator (example)

![badge](https://img.shields.io/badge/License-MIT-green.svg)

## Description
this is a javscript application that helps user create a professional read me file!

## Installation
after cloning the repository into your local machine, you will have to install the inquirer packet at version 8.2.4

## Usage
run the application and then the application will ask you for a series of questions relating to your project. After you answered all the questions it will then generate a readme file in the generated folder
